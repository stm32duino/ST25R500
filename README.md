# ST25R500

The code implements the RF/NFC abstraction layer (RFAL) APIs for the ST25R300 and ST25R500 components.

# Dependencies

The ST25R500 library requires the following STM32duino library:

* STM32duino NFC-RFAL: https://github.com/stm32duino/NFC-RFAL

## Documentation
